# 📝 To-Do List Adaptif dengan Intervensi Distraksi Digital

Aplikasi To-Do List Adaptif untuk platform Android yang dirancang khusus untuk membantu mahasiswa dan pengguna lainnya dalam melawan prokrastinasi yang disebabkan oleh distraksi digital. Aplikasi ini tidak hanya berfungsi sebagai pencatat tugas, tetapi juga secara proaktif membantu Anda untuk tetap fokus pada pekerjaan.

## ✨ Fitur Utama

  - **✅ Manajemen Tugas**: Buat, atur, dan prioritaskan daftar tugas Anda dengan mudah.
  - **⏱️ Pemantauan Aktivitas**: Aplikasi dapat memantau penggunaan aplikasi lain di perangkat Anda untuk mengidentifikasi potensi distraksi.
  - **🚫 Intervensi Proaktif**: Memberikan peringatan dan membatasi akses ke aplikasi yang Anda tandai sebagai distraksi (misalnya media sosial, game) ketika Anda memiliki tugas dengan urgensi tinggi.
  - **🧠 Adaptif**: Tingkat intervensi disesuaikan secara otomatis berdasarkan tingkat urgensi tugas yang Anda miliki.
  - **🔒 Berbasis Client-Side**: Semua data dan proses berjalan sepenuhnya di perangkat Anda untuk menjaga privasi. Tidak ada komponen server atau koneksi internet yang diperlukan.

## 🚀 Latar Belakang Proyek

Di era digital saat ini, mahasiswa seringkali kesulitan untuk fokus karena gangguan notifikasi dan hiburan dari smartphone. Fenomena ini menjadi pemicu utama prokrastinasi, yang berdampak negatif pada kinerja akademik. Aplikasi to-do list yang ada saat ini umumnya bersifat pasif dan tidak cukup untuk melawan godaan distraksi. Oleh karena itu, aplikasi ini dikembangkan untuk menjembatani kesenjangan tersebut dengan menyediakan solusi yang cerdas dan proaktif.

  - **Bahasa**: (Anda bisa isi dengan bahasa pemrograman yang digunakan, misal: Kotlin/Java/Dart)
  - **Arsitektur**: Menggunakan komponen Android seperti `Activity` dan `Service` untuk antarmuka dan proses latar belakang.

## metodologi Pengembangan

Proyek ini dikembangkan menggunakan metode **Penelitian dan Pengembangan (R&D)** dengan model **ADDIE**, yang mencakup tahapan:

1.  **Analysis (Analisis)**: Menganalisis masalah prokrastinasi dan kebutuhan pengguna.
2.  **Design (Perancangan)**: Merancang arsitektur sistem dan antarmuka pengguna (UI/UX).
3.  **Development (Pengembangan)**: Implementasi kode program untuk merealisasikan rancangan.
4.  **Implementation (Implementasi)**: Uji coba aplikasi pada perangkat target.
5.  **Evaluation (Evaluasi)**: Mengukur apakah aplikasi telah memenuhi tujuan yang ditetapkan.
